# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] — 2025-04-30

### 🚀 Initial Release — The Skill Arsenal

#### Added — Document & Data Mastery
- `docx` — Word Processing with full XML-level control
- `xlsx` — Spreadsheet manipulation with pandas & openpyxl
- `pdf` — PDF operations with pypdf, pdfplumber & reportlab
- `pptx` — Professional presentation creation & editing
- `doc-coauthoring` — 3-stage collaborative writing workflow

#### Added — Design & Visual Arts
- `frontend-design` — Production-grade UI generation
- `algorithmic-art` — Generative art with p5.js & seeded randomness
- `canvas-design` — Museum-quality static visual art
- `theme-factory` — 10 curated visual themes + custom theme generation
- `brand-guidelines` — Brand color & typography enforcement

#### Added — Development & Engineering
- `web-artifacts-builder` — Multi-component React/Vite web apps
- `webapp-testing` — Automated Playwright testing
- `claude-api` — Claude API & Anthropic SDK integration
- `mcp-builder` — Model Context Protocol server development

#### Added — Utilities & Meta-Skills
- `skill-creator` — Meta-skill for autonomous skill creation & benchmarking
- `slack-gif-creator` — Animated GIF generation for Slack
- `internal-comms` — Corporate communications formatting

#### Added — Infrastructure
- Progressive Disclosure Architecture (3-layer context management)
- Drag-and-drop `.skill` file deployment
- Built-in evaluation & benchmarking framework
- A/B testing support for skill comparison
