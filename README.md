### рҹҹў Option 1 вҖ” Drag & Drop (No-Code)

> **The fastest way to enhance your AI.** No terminal. No setup. Just drag.

| Step | Action |
|:----:|--------|
| **1** | рҹ“Ҙ **Download** вҖ” Clone or download the ZIP of this repository |
| **2** | рҹ“Ӯ **Navigate** вҖ” Open `claude (skill-zip) to upload directly/skill form/` |
| **3** | рҹ”Қ **Select** вҖ” Find your desired `.skill` file (e.g., `frontend-design.skill`) |
| **4** | рҹӘ„ **Deploy** вҖ” **Drag and drop** the file into your AI assistant interface |
| **5** | рҹ’¬ **Activate** вҖ” Ask your AI: *"Use the frontend-design skill to create a landing page"* |

<br>

### рҹ”ө Option 2 вҖ” Build from Source

> **For developers** who want to modify, inspect, or create custom skills.

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/better-than-claude-skills.git
cd better-than-claude-skills

# 2. Navigate to the target skill directory
cd frontend-design/

# 3. Make your edits to SKILL.md, scripts/, or references/
#    Customize the skill to your exact needs

# 4. Package your custom skill using the built-in bundler
python -m scripts.package_skill path/to/skill-folder

# 5. Your new .skill file is ready for deployment! рҹҡҖ
```

## рҹҸ—пёҸ Progressive Disclosure Architecture

<div align="center">
<h4>How do these skills work so flawlessly without<br>overwhelming the AI's context window?</h4>
</div>

<br>

Every skill is built on a **3-layer Progressive Disclosure Architecture**. The AI reads what it needs, *when* it needs it вҖ” nothing more.

```text
skill-name/
в”ңв”Җв”Җ SKILL.md                 # рҹ“Қ The core logic (< 500 lines)
в”Ӯ   в”ңв”Җв”Җ YAML frontmatter     # рҹ§  Trigger definition (always loaded)
в”Ӯ   в””в”Җв”Җ Markdown body        # вҡҷпёҸ High-level workflow instructions
в””в”Җв”Җ Bundled Resources        # рҹ“Ұ Loaded strictly ON-DEMAND
    в”ңв”Җв”Җ scripts/             # рҹ’» Executable Python/JS tools
    в”ңв”Җв”Җ references/          # рҹ“ҡ Deep-dive documentation
    в””в”Җв”Җ assets/              # рҹ–јпёҸ Templates, icons, fonts
```

<table>
<tr>
<td align="center" width="33%">

### рҹ§  Layer 1
**Metadata**\
*(Always Present)*

YAML frontmatter acts as a "pushy" trigger вҖ” tells the AI *when* to activate the skill.

</td>
<td align="center" width="33%">

### вҡҷпёҸ Layer 2
**Body Instructions**\
*(Loaded on Activation)*

`SKILL.md` body acts as a lightweight orchestrator вҖ” tells the AI *how* to work.

</td>
<td align="center" width="33%">

### рҹ“Ұ Layer 3
**Bundled Resources**\
*(Loaded on Demand)*

`scripts/` and `references/` are pulled into context only when explicitly needed.

</td>
</tr>
</table>
